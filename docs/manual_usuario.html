Manual de Usuario: Gestión Farmacéutica Integrada

"Excelencia en la dispensación y soberanía en el suministro regional"

1. Filosofía Institucional y Propósito

El sistema Farmacia ESEFJL representa la vanguardia en la gestión de recursos críticos para la salud amazónica. Su propósito es eliminar el riesgo de desabastecimiento en la red hospitalaria regional, garantizando que el medicamento correcto llegue al paciente correcto en el momento oportuno.

2. Operativa Integral del Sistema

Figura 1: Visión consolidada del Almacén Central.

2.1 Acceso y Perfiles

El sistema cuenta con perfiles de Regente, Gerente e IPS Municipal, cada uno con privilegios específicos para el control y despacho de insumos.

3. Farmacovigilancia (Semáforo)

El semáforo de vencimiento es la herramienta crítica para la gestión de riesgos sanitarios. La alerta roja indica que el producto debe ser retirado o priorizado para su uso inmediato.

4. Vinculación y SIAU

Cada entrega de medicamento debe estar vinculada a un paciente registrado para asegurar la trazabilidad y el cumplimiento de las normativas de auditoría médica.
